Product Management System (PMS)
Java Spring Boot AngularTS (PRIMENG) TailWind SCSS Html PostgreSQL GitHub AWS
The Product Management System (PMS) is a basic web application designed to help businesses manage their products, vendors, and communications. This project demonstrates essential skills in building and securing a full-stack application with user-friendly features.

Key Modules:

Product Management:

Product Catalog: Users can add, update, and manage a list of products. Each product includes details like name, description, price, and quantity. Search & Filter: Simple search and filter options help users quickly find specific products.

Vendor Management:

Vendor Profiles: Users can create and manage vendor information, such as names, contact details, and products they supply. Vendor Communication: Integrated with the email module to send basic email notifications to vendors.

Security Module:

User Authentication: Basic login and registration system that ensures only registered users can access the application. Role-Based Access: Different user roles (like admin and user) are implemented to control what each user can do within the system.

Email Sending Module:

Email Notifications: The system can send automated emails to vendors or users for events like new product additions or updates. Email Templates: Simple email templates can be used for sending consistent messages.
